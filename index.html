System Modules

  • N-Lateration: A mathematical brute-force spatial walk algorithm to find a 3D coordinate based on signal attenuation distances.
  • Static Fingerprinting: An indoor localization technique matching live RSSI vectors to an offline grid using K-Nearest Neighbors and Inverse Distance Weighting.
  • Hidden Markov Models (HMM): A predictive intelligence layer calculating forward and backward probabilities to anticipate user mobility state transitions.
